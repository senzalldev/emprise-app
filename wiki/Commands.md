# Commands Reference

## Chat
| Command | Description |
|---------|-------------|
| `/clear` | Reset conversation + screen |
| `/cls` | Clear screen (keep context) |
| `/name` | Name this conversation |
| `/save` | Save to history |
| `/history` | Browse past conversations |
| `/copy` | Copy last response to clipboard |
| `/code` | Browse captured code blocks |
| `/export` | Export as markdown |

## LLM
| Command | Description |
|---------|-------------|
| `/model` | Switch model (picker or direct) |
| `/profile` | Switch LLM profile |
| `/config` | Settings menu |
| `/tools` | Manage tools by category |
| `/prompts` | Prompt emphasis per role |
| `/system` | View assembled system prompt |
| `/stats` | Usage heat map per model |
| `/setup` | Add a new LLM provider |

## Developer
| Command | Description |
|---------|-------------|
| `/dev` | Environment audit + install |
| `/ps` | Background processes |
| `/stop` | Stop a background process |
| `/font` | Nerd Font setup |
| `/phi` | PHI compliance status |
| `/about` | Version, safety, practices |
| `/ask` | Quick one-off question |
| `/run` | Run shell command |

## CLI Flags
| Flag | Description |
|------|-------------|
| `emprise "question"` | Single-shot answer |
| `-c` | Continue last conversation |
| `--resume name` | Resume by name |
| `--list` | List conversations |
| `update` | Self-update |
| `install` | Reinstall |
| `-y` | Auto-approve safe tools |
| `--yolo` | Skip all confirmations |
